# 🌿 Nature Team Showcase

A modern team introduction website built with a nature-inspired glassmorphism UI and smooth animations.

🔗 **Live Website:** https://nature-team-showcase.vercel.app

---

##  Features
- 🌿 Glassmorphism UI Design
- ✨ Smooth Animations (Framer Motion)
- 📱 Fully Responsive Layout
- 👥 Team Introduction Section
- 📂 Projects Showcase
- 📬 Contact Form

---

##  Tech Stack
- Next.js
- Tailwind CSS
- Framer Motion
- TypeScript
